# Student Connect Backend

Spring Boot backend for the Student Connect platform.

## Deployment

This project is configured for deployment on Render.

### Build & Start Commands (Recommended)
- **Build Command:** `mvn clean install`
- **Start Command:** `java -jar backend/target/*.jar` (or equivalent path)

### Docker Deployment
A minimal `Dockerfile` is provided in the `backend/` directory if Docker-based deployment is preferred.
